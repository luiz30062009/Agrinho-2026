# 🌾 Agro Forte | Tecnologia no Campo
Luiz Felipe Dummer colegio estadual Costa e Silva

Site institucional moderno e responsivo da **Agro Forte**, empresa especializada em soluções tecnológicas sustentáveis para o agronegócio brasileiro.

## 📋 Descrição do Projeto

Este é um website institucional completo desenvolvido com **HTML5 + CSS3 puro**, sem frameworks. 

O design utiliza uma paleta elegante de **verde escuro** (#1b3022) e **dourado/trigo** (#c5a059), transmitindo seriedade, inovação e conexão com a terra.

### ✨ Funcionalidades

- ✅ Header fixo com navegação suave
- ✅ Seção Hero impactante com chamada para ação
- ✅ Grid de serviços com cards interativos (hover)
- ✅ Seção "Sobre" com texto institucional
- ✅ Footer com informações de contato
- ✅ Totalmente responsivo (mobile-first)
- ✅ Animações suaves e micro-interações
- ✅ Tipografia profissional (Montserrat + Roboto)

## 🚀 Como Executar o Projeto

### Opção 1: Abrir diretamente (mais simples)
1. Baixe/clone este repositório
2. Abra o arquivo **`index.html`** no seu navegador

### Opção 2: Usando VS Code + Live Server (recomendado)
1. Instale a extensão **Live Server**
2. Clique com botão direito no `index.html`
3. Selecione **"Open with Live Server"**

O site abrirá automaticamente em `http://localhost:5500`
